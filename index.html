Inner Join

INNER JOIN is a function that will return the intersection between tables using the condition we give to the function.
Below is the syntax for INNER JOIN: 
                SELECT <column(s) name for new table> 

                FROM <table 1 name> 

                INNER JOIN <table 2 name> 

                ON <condition>
We can find the intersection between 3 or more tables by using nested INNER JOIN: 
                SELECT <column(s) name for new table> 

                FROM (((<table 1 name> 

                INNER JOIN <table 2 name>) 

                INNER JOIN <table 3 name>) 

                INNER JOIN ...) 

                ON <condition>
We use INNER JOIN when we want to merge two data sets that are related to each other through a variable.
For example, we have 2 tables, Mealplan and Name:
StudentID MealPlan
000001 10 meals
000002 7 meals
000003 14 meals
StudentID StudentName
000001 Andy
000002 Bob
000003 Charlie
and we use the INNER JOIN: 
                SELECT Mealplan.MealPlan, Name.StudentName 

                FROM Mealplan 

                INNER JOIN Name

                ON Mealplan.StudentID = Name.StudentID
and we will have this:
MealPlan StudentName
10 meals Andy
7 meals Bob
14 meals Charlie

Data Scientist

What is a data scientist?

First, a data scientist is a scientist. And scientist makes hypothesis, carry out experiments and then discover new things. Like chemist deals with chemicals, data scientist deals with data.

How do they do that?

Data scientist are just human. They can't look at numbers and then come to some conclusion about a bunch of numbers. First they will gather all the data, and then use algorithms to integrate that data, or create visualizations to see the trend. To do this, they need a strong foundation in mathematics, statistics, computer science,...

Data scientist vs. Computer scientist

A data scientist and a computer scientist have many things in common. They both have to know about algorithms and programming language. But while computer scientist answers the question what is the result, data scientist answers the question why is the result like this. Therefore, data scientists need more knowledge in algorithms and tools that are highly data-oriented such as machine learning or neural network.
For example, if we were to make an application, a game for instance, a data scientist would tell what should the game include to attract the most people through a lot of analysis, and a computer scientist would create that game.
